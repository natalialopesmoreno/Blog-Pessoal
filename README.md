# Blog-Pessoal

Criação de um Blog Pessoal utilizando a linguagem Java, 
o Framework Spring Tools, 
a arquitetura Spring MVC 
e como dependências:

-MYSQL
-JPA
-Spring Verification
-Spring DevTools
-Spring Web
-Thymleaf
-Swagger 
-Commons Codec

Para segurança e criptação de dados foi utilizado Spring Security
